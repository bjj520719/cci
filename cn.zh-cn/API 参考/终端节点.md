# 终端节点<a name="cci_02_2003"></a>

终端节点（Endpoint）即调用API的**请求地址**，不同服务不同区域的终端节点不同，您可以从[地区和终端节点](https://developer.huaweicloud.com/endpoint)中查询所有服务的终端节点。

云容器实例的终端节点如下表所示，请您根据业务需要选择对应区域的终端节点。
